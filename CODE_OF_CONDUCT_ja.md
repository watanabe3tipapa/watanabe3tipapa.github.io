# コントリビューター行動規範

## 私たちの約束

私たちは、オープンで歓迎されるコミュニティを育成するために、貢献者およびメンテナーとして、年齢、体型、障害、民族性、性別、経験レベル、教育、社会経済的地位、国籍、外見、人種、宗教、性的アイデンティティと指向に関係なく、すべての人にとってハラスメントのない環境を維持することを誓います。

## 標準

歓迎される行動：

* 共感と親切を示す
* 異なる意見を尊重する
* 建設的なフィードバックを受け入れる
* 過ちに対して責任を取り、影響を受けた人々に謝罪する
* コミュニティ全体にとって最善なことに焦点を当てる

容認されない行動：

* 性的な言葉や画像の使用
* 荒らし、侮辱的コメント、個人攻撃
* 公的または私的なハラスメント
* 明示的な許可なく他者の個人情報を公開する行為
* その他、プロフェッショナルとして不適切と合理的にみなされる行為

## 執行責任

プロジェクトメンテナーは、行動規範の基準を明確にし、執行する責任を負います。

## 適用範囲

この行動規範は、すべてのコミュニティスペースに適用されます。

## 報告

虐待、ハラスメント、その他不適切な行為はプロジェクトメンテナーに報告してください。すべての苦情は迅速にレビューされます。

## 帰属

この行動規範は [Contributor Covenant][homepage] バージョン2.1 を基に作成されています。
