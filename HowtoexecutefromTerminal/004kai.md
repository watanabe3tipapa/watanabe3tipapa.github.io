- ターミナルから実行
    
    任意のディレクトリに移動します（事例：ホームディレクトリ）
    
    ```bash
    cd
    ```

    ホームディレクトリに対象となるJPEGファイルを保存しておきます  


    次のコマンドによりJPEGファイル（事例：IMG_7652.jpeg）のEXIF情報が削除されます
    ```bash
    jhead -purejpg IMG_7652.jpeg
    ``` 

    次のコマンドにより確認できます
    ```bash
    jhead IMG_7652.jpeg
    ```

    
    >>> 結果
    コマンドライン上でJPEGファイルのEXIF情報を削除できました。
